Les colonnes sous Bootstrap

Créer trois colonnes avec les effets suivants :

  • Si le média est un téléphone, la colonne occupe la totalité de l'écran.
  • Tant que la largeur est inférieur à celle d'un ordinateur classique, les colonnes occupent un espace de 8.
  • Au dela les colonnes occupent un espace de 4.
